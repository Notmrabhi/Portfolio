Music Bot

Advance Discord bot that plays music, kick/ban , sets deadlines, timers and many more features (Discord, APIs, Bots) Used Spotify, SoundCloud And Youtube to search for music and Discord API to search gifs and emojis, used Guild Functionality of Discord.js to send DMs to remind users of deadlines set by them. Streaming in more then 30 servers. Tech Stack: Javascript, Discord.Js, API and variety of different commands and features including a play command that can load single songs, playlists, or search generally from the internet, a paginated queue command, pause/resume commands, info command with a progress bar.

Bhagalpuri Silk

Full stack MERN e-commerce application shopping app with cart, ordering, payment gateway etc. features. This project is based on my city Bhagalpur which is know as the silk city of india. Here we have all categories of silk sarees listed where you can sort according to the color and price, also you can add sarees to you favourite list.
